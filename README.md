# Torah Daily | תורה יומי

A mobile-ready Progressive Web App (PWA) for daily Torah study.  
Pulls live Hebrew + English text from Sefaria · Text-to-speech audio · AI commentary via Claude.

---

## Files
- `index.html` — the entire app (one self-contained file)
- `manifest.json` — PWA metadata (enables "Add to Home Screen")
- `sw.js` — service worker for offline caching

---

## Hosting options (choose one)

### Option A: GitHub Pages (free, permanent URL)
1. Create a free account at github.com
2. Click **New repository** → name it `torah-daily` → Public
3. Upload all three files
4. Go to Settings → Pages → Source: "Deploy from branch" → `main` / `/ (root)` → Save
5. Your app will be live at `https://YOUR-USERNAME.github.io/torah-daily/`
6. On your phone: open that URL in Safari (iOS) or Chrome (Android) → "Add to Home Screen"

### Option B: Netlify Drop (fastest — drag and drop)
1. Go to https://app.netlify.com/drop
2. Drag the `torah-daily` folder onto the page
3. You get an instant URL like `https://random-name-123.netlify.app`
4. Open on phone → Add to Home Screen

### Option C: Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. `cd torah-daily && vercel --prod`
3. Follow prompts → instant live URL

---

## Installing on your phone

### iPhone (Safari)
1. Open your hosted URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add** — it appears like a native app

### Android (Chrome)
1. Open your hosted URL in **Chrome**
2. Tap the **⋮ menu** → **Add to Home screen**
3. Tap **Add** — or Chrome may show a banner automatically

## Features
- All 54 weekly parashot (Genesis through Deuteronomy)
- Each parasha divided into 7 traditional aliyot
- Side-by-side Hebrew (Frank Ruhl Libre) and English (JPS via Sefaria)
- Text-to-speech in English or Hebrew
- AI commentary generated by Claude (draws on Rashi, Ramban, modern insights)
- Auto-detects the current week's parasha
- Works offline (app shell cached; text fetched fresh)
- Installable on iPhone or Android

- Hebrew text and English translation provided by Sefaria (sefaria.org) under CC BY-SA
